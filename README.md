# Virtual-CPU
Руководство по языку программирования для мини-Процессора
Общие сведения
Этот язык предназначен для программирования простого виртуального процессора. Программа состоит из последовательности команд, каждая из которых управляет регистрами и памятью процессора.

Основные команды:
MOV <регистр> <значение/регистр> — копирует значение или содержимое регистра в указанный регистр.
ADD <регистр> — прибавляет содержимое указанного регистра к аккумулятору (ACC).
INPUT <регистр> — запрашивает ввод числа пользователя и сохраняет его в указанный регистр.
PRINT <регистр или значение> — выводит содержимое регистра или значение.
HALT — завершает выполнение программы.
Структура команд
1. MOV
asm

MOV Rn значение
MOV Rn Rm
MOV R1 10 — запишет число 10 в регистр R1.
MOV R1 R2 — скопирует значение из регистра R2 в R1.
2. ADD
asm

ADD Rn
Прибавляет содержимое регистра Rn к аккумулятору ACC.
Например: ADD R1 — прибавит значение R1 к ACC.
3. INPUT
asm

INPUT Rn
Запрашивает число у пользователя и сохраняет его в регистр Rn.
4. PRINT
asm

PRINT Rn
PRINT значение
Выводит значение регистра или число.
5. HALT
asm

HALT
Останавливает выполнение программы.
Пример программы
asm

MOV R1 0          // Инициализация R1 нулём
MOV R2 0          // Инициализация R2 нулём
INPUT R1          // Ввод числа в R1
INPUT R2          // Ввод числа в R2
MOV ACC R1        // Перенос R1 в аккумулятор
ADD R2           // Прибавление R2 к аккумулятору
PRINT ACC        // Вывод результата
HALT             // Конец программы
Руководство по использованию
Каждая команда занимает одну строку.
Команды чувствительны к регистру: используйте MOV, ADD, INPUT, PRINT, HALT.
Указатели регистров: R1, R2, ..., R16.
Значения: целые числа.
