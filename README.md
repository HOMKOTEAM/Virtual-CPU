Руководство по языку программирования для мини-процессора

Этот язык создан для программирования простого виртуального процессора. Программа состоит из команд, которые управляют регистрами и памятью.

Основные команды:

1. MOV <регистр> <значение/регистр>
Копирует значение или содержимое регистра в указанный регистр.
* MOV Rn значение
* MOV Rn Rm
  Пример:
* MOV R1 10
  Запишет число 10 в регистр R1.
* MOV R1 R2
  Скопирует значение из регистра R2 в R1.
2. ADD <регистр>
Прибавляет содержимое указанного регистра к аккумулятору (ACC).
Пример:
* ADD R1
  Прибавит значение регистра R1 к ACC.
3. INPUT <регистр>
Запрашивает у пользователя ввод числа и сохраняет его в указанный регистр.
4. PRINT <регистр или значение>
Выводит значение регистра, число или текст.
5. HALT
Завершает выполнение программы.

Структура команд:

1. MOV
Пример:
* MOV Rn значение
* MOV Rn Rm
2. ADD
Пример:
* ADD Rn
3. INPUT
Пример:
* INPUT Rn
4. PRINT
Пример:
* PRINT Rn
* PRINT значение
5. HALT

Руководство по использованию:

* Каждая команда занимает одну строку.
* Команды чувствительны к регистру: используйте MOV, ADD, INPUT, PRINT, HALT.
* Указатели регистров: R1, R2, ..., R16.
* Значения: целые числа.
